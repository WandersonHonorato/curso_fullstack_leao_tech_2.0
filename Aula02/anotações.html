Terminal (Power Shell)

ls = verificar as pastas existentes cd = navegar na pasta cd .. = voltar na pasta anterior

Comandos Git (Git Bash)

git init = iniciar repositório git add . = adicionar todos os comentários git commit -m "comentário" = adcionar commit (comentário) git push = enviar modificações para o repositorio remoto git push force = forçar envio modificações para o repositorio remoto git status = verficiar o ultimo commit git log --oneline = verificar todos os commits realizados git pull = atualizar o seu repositório local com as últimas alterações feitas no repositório remoto
